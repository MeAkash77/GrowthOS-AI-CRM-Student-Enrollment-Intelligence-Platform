# Assets Folder

This folder contains images and media files for GreenCRM.

## 📁 Structure

```
assets/
├── images/
│   ├── landing-page.png    # Landing page screenshot
│   └── dashboard.png       # Dashboard screenshot
└── README.md               # This file
```

## 📝 Usage

- **landing-page.png**: Screenshot of the main landing page showing the hero section, features, and overall design
- **dashboard.png**: Screenshot of the main dashboard showing AI insights, charts, and CRM interface

## 🖼️ Adding New Images

1. Place images in the appropriate subfolder
2. Use descriptive names
3. Optimize for web (PNG/JPEG, reasonable file size)
4. Update this README if adding new categories
